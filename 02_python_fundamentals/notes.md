# 第2章 学習メモ

## 学習期間
2026/03/01 〜 2026/03/08

## 理解できたこと
- int・float・str間の型変換（int()・float()・str()）
- 文字列のインデックスアクセスとスライス
- リストの作成・追加（append）・変更・長さ（len）
- mathモジュールの関数（sqrt・pi・floor・ceil）
- randomモジュールでランダムな値を生成できる
- importとfrom〜importの違い

## 詰まったこと・疑問
- スライスのendは含まれない（[0:3]は0・1・2）
- int(3.99)が3になる（四捨五入ではなく切り捨て）
- random.shuffle()は戻り値がなく元のリストを直接変える

## 気づき・応用アイデア
- math.sqrt()で選手間の距離計算ができる
- random.choice()でランダムに選ぶ抽選ツールが作れる
- スライスを使えばリストの上位N件だけ取り出せる
- as 別名でモジュールを別名でインポートできる
